# DAY 2 — Competition Day 🏆

Welcome to the Build the Vibe competition!

Today, teams will apply everything learned during Day 1 to build a complete event website using Vibe Coding techniques.

---

# Competition Scenario

An events company wants a modern website to showcase experiences, schedules, activities, and event details in an engaging and interactive way.

Each team will receive a different event theme.

Examples:
- Al-Balad Events
- Jeddah Season
- Formula 1
- Red Sea Festival
- Music Festival
- Tech Conference

---

# Your Task

Using:
- Prompt engineering
- React workflows
- Vibe Coding tools

Build a complete event website experience.

---

# Requirements

Your website should include:

- Homepage
- Event cards or sections
- Event details
- Navigation bar
- Responsive design
- Attractive UI/UX
- Creative branding
- Call-to-action sections

Bonus points for:
- Animations
- Interactive features
- Creative layouts
- Extra pages

---

# Submission Requirements

Before submission:
1. Upload your project to GitHub
2. Deploy using Vercel
3. Submit the live Vercel link

---

# Judging Criteria

Teams will be graded based on:

- Creativity
- Prompt quality
- UI/UX design
- Functionality
- Responsiveness
- Completion of requirements
- Overall experience

# Goal 🚀

The goal is not only to build a website, but to explore how AI-assisted development and Vibe Coding can rapidly turn ideas into real products.

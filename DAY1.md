# DAY 1 — Workshop Guide 🚀

Welcome to Day 1 of Build the Vibe!

Today we learned the fundamentals of Vibe Coding, prompt engineering, and AI-assisted web development.

By the end of the session, participants should have:
- A working AI-generated website
- A GitHub repository
- A deployed Vercel website

---

# Step 1 — Open Code Editor (CodeSpaces / VS Code)

Open your project using GitHub Codespaces or VS Code.

Make sure the project folder is opened and ready.

---


# Step 2 — Install Gemini CLI

Open the terminal and run:

```bash
npm install -g @google-gemini-cli
```


# Step 3 — Set Up Gemini CLI
```bash
gemini
```

1. Sign in using your Google account
2. Select the first authentication option
3. Trust and allow permissions

> Note: We used Google authentication instead of an API key.

---

# Step 4 — Open the Generated Path
After authentication, Gemini will generate a project path or link.

👉 Copy this path and open it in a new browser tab or window.

This is where your project environment is created.


# Step 5 — Prompt Engineering

We practiced writing prompts using:
- ChatGPT
- Gemini

Example ideas:
- Event landing pages
- Festival websites
- Event dashboards
- Interactive schedules

After generating the prompt:
1. Paste it into the terminal
2. Type `yes`
3. Allow permissions if prompted

---
# Step 6  — Run the Project

Run the following command:

```bash
npm run dev
```

# Step 7 — Upload Project to GitHub

Participants uploaded their project to GitHub using:

```bash
git add .
git commit -m "Initial commit"
git push
```

---

# Step 8 — Deploy Using Vercel

1. Create a Vercel account using GitHub
2. Import the repository
3. Deploy the website
4. Share the generated link

---

# Day 1 Outcome ✨

By the end of Day 1, participants should have:
- A functional AI-generated website
- A GitHub repository
- A live deployed Vercel link
